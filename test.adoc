== USECASE1: Top 5 ingredients

// REMOVEME: Describe what this use-case is solving and how the query represents that

[source,cypher]
----
MATCH (a:Food)-[:IS_INGREDIENT]->(r:Recipe)
RETURN a.name, count(*) as freq, collect(r.name) as recipes
ORDER BY freq DESC
LIMIT 5
----
